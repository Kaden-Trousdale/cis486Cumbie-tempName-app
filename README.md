# 🍽️ Seal Chef's Questionable Recipes

## Purpose

A full-stack CRUD web application for sharing and discussing recipes with image uploads, likes, and comments. Demonstrates a complete data round trip from front-end to MongoDB database with full CRUD operations and user interactions.

## Technology Stack

**Backend:** Node.js, Express.js, MongoDB  
**Frontend:** HTML5, CSS3, Bootstrap 5, jQuery  
**DevOps:** GitHub Actions (CI/CD), Render, Google Cloud Platform
